# Cabin Classes

Four cabin classes are available across the fleet. Availability depends on ship class — the Helion only offers Drift and Orbit; the Lunara starts at Orbit. Cabin selection happens after cryostasis selection, because Full Cryo passengers skip cabin amenities entirely and some classes are incompatible with certain cryo options.

:::note Placeholder Names
These cabin class names are placeholders pending final world-building decisions. The tier structure, pricing multipliers, and feature sets are final.
:::

## Drift Class

**Price Multiplier:** 1.0× (base)  
**Private Cabin:** No  
**Viewport:** None  
**Dining:** Communal  
**Cryo:** All options

Shared sleeping quarters with bunk-style berths. Access to communal lounges, communal dining, and shared recreation spaces. For passengers who need to get somewhere and don't need to be comfortable doing it.

On a Helion-class shuttle, Drift is the standard experience — the journey is a few hours, the distinction between cabin classes barely matters. On a Solaris-class voyage to Calyx, Drift over 192 days is a different proposition — social, communal, and for passengers who prefer hostel energy to hotel energy.

---

## Orbit Class

**Price Multiplier:** 1.8×  
**Private Cabin:** Yes  
**Viewport:** Screen (external camera feed)  
**Dining:** Standard  
**Cryo:** All options

A private cabin with a bed, personal storage, mid-tier dining, and recreation access. The viewport screen shows a real-time external feed rather than a physical hull port — the view is authentic, the glass is not.

The standard experience for most travellers. The privacy premium is modest relative to the amenity uplift, making Orbit the default recommendation for journeys over three days.

---

## Apex Class

**Price Multiplier:** 3.2×  
**Private Cabin:** Yes  
**Viewport:** Real hull viewport  
**Dining:** Dedicated — priority access and premium menu  
**Cryo:** All options

A premium suite with a real hull viewport — physical glass between the passenger and space. Larger than Orbit, with dedicated dining access, priority boarding at waypoints, and exclusive lounge access.

The viewport distinction matters more than it sounds. On a long voyage, the difference between looking at a screen and looking through actual glass at actual space is significant. Apex passengers consistently report the viewport as the most valued feature.

---

## Helix Class

**Price Multiplier:** 6.0×  
**Private Cabin:** Yes  
**Viewport:** Private observation bubble  
**Dining:** Exclusive — dedicated steward, captain's table access  
**Cryo:** Conscious Voyage only  
**Ship Classes:** Tethys, Lunara, Solaris only

The top tier. A full suite with a personalised steward, private observation bubble — a transparent pod protruding from the hull offering 270° unobstructed stellar views — and captain's table access. All standard add-on packages are included.

Helix is Conscious Voyage only. The experience is the product. Placing a Helix passenger in cryostasis for the duration would defeat the entire purpose of the class.

The observation bubble is the defining feature: a separate pressurised compartment attached to the suite, accessible through an airlock, where the passenger sits in a transparent pod with the void in every direction. Nothing between them and deep space but a few centimetres of glass.

:::info Helix Class and Cryostasis
Helix Class is incompatible with Full Cryo and Cryo Intervals. Passengers booking Helix are committing to the Conscious Voyage experience. If you want cryo, choose Apex or below.
:::
