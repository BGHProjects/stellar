# Packages & Add-Ons

Add-ons are selected after cabin class, on the Packages page. They are grouped into five categories. **Journey Protection** and **Expedition Extras** are available to all passengers regardless of cryostasis choice. The remaining categories — Dining, Recreation, and Entertainment — are only available to Conscious Voyage and Cryo Intervals passengers.

Full Cryo passengers see these categories grayed out in the booking flow.

---

## Journey Protection

Available to all passengers, all cryo options.

| Add-On                          | Description                                                                                       |
| ------------------------------- | ------------------------------------------------------------------------------------------------- |
| Radiation Shield Insurance      | Covers medical costs from unexpected solar event exposure                                         |
| Asteroid Deviation Coverage     | Ship reroutes around unexpected debris fields. Particularly relevant for Scatter-crossing routes. |
| Emergency Evacuation Policy     | Covers cost of an emergency transfer vessel if voyage is interrupted                              |
| Journey Interruption Protection | Refunds and rebooking if voyage is significantly delayed or cancelled                             |

Asteroid Deviation Coverage is highlighted in the booking flow when the selected route crosses the Scatter — the insurance product exists because of that specific risk.

---

## Dining Upgrades

Available to: Conscious Voyage and Cryo Intervals passengers.

| Add-On                  | Notes                                                                                      |
| ----------------------- | ------------------------------------------------------------------------------------------ |
| Chef's Table Access     | Premium rotating menu by the ship's head chef. Bookable per evening.                       |
| Cultural Cuisine Weeks  | Themed dining representing destination world culinary traditions. Scheduled during voyage. |
| Private In-Cabin Dining | Available to Apex and Helix Class passengers only.                                         |

## Recreation & Wellbeing

Available to: Conscious Voyage and Cryo Intervals passengers.

| Add-On                       | Notes                                                            |
| ---------------------------- | ---------------------------------------------------------------- |
| Zero-G Spa Access            | Wellness deck using the ship's variable gravity sections         |
| Stellar Observatory Sessions | Reserved time in the main observation dome with guided narration |
| Fitness & Training Suite     | Gravity-gym designed for long-voyage physical conditioning       |

---

## Entertainment

Available to: Conscious Voyage and Cryo Intervals passengers.

| Add-On                    | Notes                                                                           |
| ------------------------- | ------------------------------------------------------------------------------- |
| Immersive Reality Suite   | Bookable sessions in a full-environment simulation room                         |
| Ship Library & Archive    | Digital archive of Taunor system history, culture, and science                  |
| Live Performance Schedule | Musicians, speakers, theatrical performances. Available on select voyages only. |

The Live Performance schedule is voyage-specific — it only appears in the booking flow if the selected voyage has a scheduled performance programme.

---

## Expedition Extras

Available to all passengers, all cryo options.

| Add-On                   | Notes                                                                |
| ------------------------ | -------------------------------------------------------------------- |
| Shore Excursion          | Pre-booked surface or orbital excursions at destination or waypoints |
| Private Arrival Transfer | Pre-arranged shuttle from spaceport to onward destination            |
| Premium Cargo Hold       | Additional hold allocation beyond standard luggage allowance         |
| Spacewalk Experience     | Walk the ship's exterior under crew supervision                      |

### The Spacewalk Experience

The Spacewalk is available on Tethys, Lunara, and Solaris class vessels. It is segment-restricted — not available near high-radiation zones, the Vareth magnetosphere, or active debris fields. The booking flow checks the voyage segment and enables the add-on accordingly.

The Spacewalk is one of the most premium optional experiences in the booking flow — not because of price, but because it has no terrestrial equivalent. Walking the exterior of a ship in deep space, with nothing between you and the void, is an experience that a very small number of humans in the fictional history of the Taunor system have had access to. Stellar offers it as an add-on.
