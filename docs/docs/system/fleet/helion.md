# Helion-class

**Nickname:** The Sprint  
**Manufacturer:** Voss-Kiran Transit Systems  
**Introduced:** 2761  
**In Service:** 24 active vessels  
**Cruise Speed:** 0.10 AU/day

> _Fast. Direct. No ceremony._

## Overview

The smallest and fastest vessel class in the Stellar fleet. Built for efficiency on short-to-medium routes where passengers want to arrive, not experience the journey. Everything that did not contribute to speed and low operational cost was removed.

Helion-class vessels operate the inner shuttle circuits that keep the Vareth moon network and the Calyx system connected on near-daily schedules. Their crews are considered among the most technically skilled in the fleet — the magnetosphere approach to Kalos, threading between Vareth's radiation bands, requires precision that the Helion's size and manoeuvrability make possible.
