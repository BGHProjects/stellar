# The Fleet

Stellar operates four vessel classes across the Taunor network. Unlike commercial airlines — where the same 737 might fly any route — each ship class has a defined operational profile. The Helion is built for speed on short inner-system hops; the Solaris is built for comfort on voyages measured in months.

Multiple vessels of each class are flying simultaneously. You book a flight number on a class, not a specific named ship.

## Speed and Journey Time

Ship speed is expressed in AU per day. Journey durations depend on both ship speed and the orbital separation between origin and destination at the time of departure.

At average orbital separation, indicative journey times:

| Route                        | Helion   | Tethys  | Lunara   | Solaris  |
| ---------------------------- | -------- | ------- | -------- | -------- |
| Aethon → Kalos (~2.2 AU avg) | 22 days  | 40 days | 52 days  | 88 days  |
| Aethon → Calyx (~4.8 AU avg) | 48 days  | 87 days | 114 days | 192 days |
| Kalos → Thal (~0.012 AU)     | ~3 hours | —       | —        | —        |

These are averages. Actual durations vary by ±30–40% depending on the orbital window at departure.

## Choosing a Class

**Helion** — you need to get somewhere short in the Vareth or Calyx moon systems, and you want the fastest option. No cryo, minimal amenities. Pure transport.

**Tethys** — the standard choice for most routes. Comfortable, reliable, available on both direct and gravity assist routes. The workhorse.

**Lunara** — the journey is part of why you're going. You want the observation deck, the views through the Scatter, the approach to Vareth's rings. No Full Cryo available — you're expected to be awake for this.

**Solaris** — you're going to Calyx or Vael, the journey is measured in months, and you want to actually live during those months. Or you want to sleep through it in a cryo pod the size of a room. Either way, the Solaris is the only ship that can do it comfortably at that duration.
